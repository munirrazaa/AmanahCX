Getting Started — From Invite Email to First Login

Before a new team member can use AmanahCX, a Tenant Admin invites them. This section covers exactly what happens from the moment an admin sends an invite to the moment the new user is logged in.

Step 1 — Admin invites a user

1
Go to Users → Invite User Fill in the person's full name, email address, department, role, and (optionally) their line manager.
The form clearly states: "An invitation email will be sent with a password-setup link (valid 7 days)." Both the department and role fields are required — the admin must decide these upfront, not the new user.

Step 2 — New user receives the email

The person invited receives an email containing a one-time password-setup link. This link is only valid for 7 days — if it's not used in time, the admin needs to resend the invite.

Step 3 — New user sets their password

Clicking the link opens the password-setup page. The new user chooses a password (minimum 8 characters) and confirms it.

Step 4 — First login

Once the password is set, the user signs in from the login page using three things: their workspace name (e.g. haier-electronics), their email address, and the password they just created.

After signing in, the user lands on the page appropriate to their role — a Tenant Admin lands on the Admin Dashboard (covered next).
